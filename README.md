# Starcks × MarathonBet — Proposta commerciale

Pagina web interattiva che presenta le due proposte di partnership tra Starcks e MarathonBet:

- **Offerta 01 — Starcks 360**: sponsorship esclusiva sull'intero ecosistema (community + gamification).
- **Offerta 02 — Social di Marathon**: costruzione di asset social proprietari da zero.

La pagina permette di esplorare nel dettaglio ogni voce, confrontare le due strade e approfondire i contenuti.

## Come aggiornare la pagina

La pagina è un singolo file `index.html`. Per modificarla:

1. Apri `index.html` nel repo e clicca l'icona di modifica (la matita), oppure carica una nuova versione con **Add file → Upload files**.
2. Salva con **Commit changes**.
3. GitHub Pages ripubblica la pagina automaticamente in 1–2 minuti. L'URL resta invariato.
